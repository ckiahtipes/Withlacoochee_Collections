The goal of this project is to use the data availble from the Atlas of Florida Plants to guide collection efforts in the field. In the most general sense, I want to figure out a rough methodology for taking past collection information and
using it to identnify potential gaps in our knowledge of plant community composition or the distribution of Florida's plants.

We are using the Withlacoochee State Forest and its various co-managed areas as a first case study because it's beautiful and it's fun to work there.
